# apt_repositories

This role allows you to download repository keys and configure
repositories in `/etc/apt/sources.list` or `/etc/apt/sources.list.d`
using the "One-Line-Style Format" or the "Deb822-style Format".

[Explanation of the DEB822 Source Format](https://repolib.readthedocs.io/en/latest/deb822-format.html)
